    Message Structure¶

    Basic Format¶

    {L[n]
  item_1
  item_2
  ...
  item_n
}

or

{L[n]
  GET(group_name.message_name1)
  GET(group_name.message_name2)
  ...
  GET(group_name.message_name[n])
}
    • L[n]: List containing n items
    • item: Individual data element of any supported type
