## System Requirements

| Requirement | Specification |
|-------------|---------------|
| **OS** | Windows 11+ / Windows Server 2003+ |
| **Memory** | Minimum 32 GB |
| **.NET Runtime** | [.NET SDK 8.0.403 (Windows x64)](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer) |

---

## Installation

1. Download the installer from the link above.
2. Run `Setup.exe` and follow the on-screen instructions.
3. Install [.NET SDK 8.0.403](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer) if not already present.
4. Launch **PortDIC** from the Start menu or desktop shortcut.

---

## VSCode Extension

Install the PortDIC extension directly from the VSCode Marketplace:

- Open VSCode → Extensions (`Ctrl+Shift+X`)
- Search for **PortDIC**
- Click **Install**

Or install via the command line:

```bash
code --install-extension portroconn.portdic
```
