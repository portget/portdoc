# ダウンロード

## PortDIC インストーラー

| 名前 | バージョン | OS | 安定版 | リンク |
|------|----------|----|--------|--------|
| Windows 用バイナリ | v1.2.2 | Windows x64 | Yes | [v1.2.2-win-installer](pathname:///file/Setup.zip) |

## システム要件

| 要件 | 仕様 |
|------|------|
| **OS** | Windows 11+ / Windows Server 2003+ |
| **メモリ** | 最低 32 GB |
| **.NET ランタイム** | [.NET SDK 8.0.403 (Windows x64)](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer) |

---

## インストール手順

1. 上のリンクからインストーラーをダウンロードします。
2. `Setup.exe` を実行し、画面の指示に従います。
3. [.NET SDK 8.0.403](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer) がインストールされていない場合は先にインストールします。
4. スタートメニューまたはデスクトップのショートカットから **PortDIC** を起動します。

---

## VSCode 拡張機能

VSCode Marketplace から PortDIC 拡張機能を直接インストールします：

- VSCode を開く → 拡張機能（`Ctrl+Shift+X`）
- **PortDIC** を検索
- **インストール** をクリック

またはコマンドラインでインストール：

```bash
code --install-extension portroconn.portdic
```
