# PortDIC へようこそ

`"あらゆるドキュメントはモデルになれる。"`
**「ドキュメントはシンプルに。アプリケーションは素早く。」** というモットーのもと設計された Port は、
ドキュメントを作成するだけで OPC UA、SECS、MQTT、REST API、スクリプトベースのフロー制御を実装できます。
また、Port パッケージを通じて、複数のプログラミング言語で再利用可能なライブラリを作成することもできます。

ドキュメントの [ライセンス](license.md) をご確認ください。
