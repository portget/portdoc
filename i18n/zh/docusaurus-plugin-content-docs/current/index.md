# 欢迎使用 PortDIC

`"任何文档都可以成为模型。"`
Port 以 **"文档化繁为简，应用开发提速"** 为设计理念，
让用户仅通过创建文档即可实现 OPC UA、SECS、MQTT、REST API 及基于脚本的流程控制。
此外，借助 Port 包，用户还可以为多种编程语言创建可复用的库。

请查阅文档 [许可证](license.md)
