|------|------|

| **内存** | 最低 32 GB |
| **.NET 运行时** | [.NET SDK 8.0.403 (Windows x64)](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer) |

## 安装步骤

1. 从上方链接下载安装程序。
2. 运行 `Setup.exe` 并按照屏幕提示操作。
3. 如尚未安装，请先安装 [.NET SDK 8.0.403](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer)。
4. 从开始菜单或桌面快捷方式启动 **PortDIC**。

---

## VSCode 扩展

直接从 VSCode 市场安装 PortDIC 扩展：

- 打开 VSCode → 扩展（`Ctrl+Shift+X`）
- 搜索 **PortDIC**
- 点击**安装**

或通过命令行安装：

```bash
code --install-extension portroconn.portdic
```
