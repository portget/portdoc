# 다운로드

## PortDIC 설치 파일

| 이름 | 버전 | OS | 안정 버전 | 링크 |
|------|------|----|-----------|------|
| Windows용 바이너리 | v1.2.2 | Windows x64 | Yes | [v1.2.2-win-installer](pathname:///file/Setup.zip) |

---

## 시스템 요구사항

| 요구사항 | 사양 |
|----------|------|
| **OS** | Windows 11+ / Windows Server 2003+ |
| **메모리** | 최소 32 GB |
| **.NET 런타임** | [.NET SDK 8.0.403 (Windows x64)](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer) |

---

## 설치 방법

1. 위 링크에서 설치 파일을 다운로드합니다.
2. `Setup.exe`를 실행하고 화면의 안내를 따릅니다.
3. [.NET SDK 8.0.403](https://dotnet.microsoft.com/ko-kr/download/dotnet/thank-you/sdk-8.0.403-windows-x64-installer)이 설치되어 있지 않다면 먼저 설치합니다.
4. 시작 메뉴 또는 바탕화면 바로가기에서 **PortDIC**을 실행합니다.

---

## VSCode 확장

VSCode 마켓플레이스에서 PortDIC 확장을 직접 설치합니다:

- VSCode 실행 → 확장(`Ctrl+Shift+X`)
- **PortDIC** 검색
- **설치** 클릭

또는 명령줄로 설치:

```bash
code --install-extension portroconn.portdic
```
